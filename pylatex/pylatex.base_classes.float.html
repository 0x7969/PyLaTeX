pylatex.base_classes.float¶

This module implements the classes that deal with floating environments.

class Float(position=None, separate_paragraph=True, **kwargs)¶

Bases: pylatex.base_classes.containers.Environment

A class that represents a floating environment.

Parameters:
  • position (str) – Define the positioning of a floating environment, for instance 'h'. See the references for more information.
  • separate_paragraph (bool) – By default floats are positioned inside a separate paragraph. Setting this to option to False will change that.

References

  • https://www.sharelatex.com/learn/Positioning_of_Figures
